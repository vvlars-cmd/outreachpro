Personalised Video Thumbnails

Add a fake "video" thumbnail to any cold email — with the lead's name personalised directly on the image. Click rates increase by 3× on average because people are naturally drawn to play buttons.

How It Works

OutreachPro generates an SVG image showing a dark background, a play button, and the lead's name and city personalised as text. The image is base64-encoded and embedded directly in the email — no external image hosting needed, no broken images.

Generating a Thumbnail

  1. Go to Email Composer → Video Thumbnail
  2. Select a lead (or generate for all leads in batch)
  3. Enter your headline text (e.g. "80% faster bike cleaning — for {{name}}")
  4. Enter CTA text (e.g. "Watch the 2-minute demo →")
  5. Choose background colour
  6. Click Generate
  7. Copy the embed HTML and paste it into your email body

Batch Generation

Generate thumbnails for all leads in your campaign at once. Go to Email Composer → Batch Thumbnails, configure the design, and click Generate All. Each thumbnail is unique — the lead's name and city are personalised on every image.

API

POST /api/video-thumbnail
  Body: { leadName, leadCity, headline, cta, bgColor, textColor }
  → { svg, dataUri, embedHtml }

POST /api/video-thumbnail/batch
  Body: { leads[], headline, cta, bgColor }
  → { count, thumbnails[] }
