Sales Coaching Dashboard

The coaching dashboard gives managers a real-time view of pipeline health, team performance, best-performing campaigns, A/B test winners, and overdue follow-ups. Claude generates a specific coaching insight based on the actual numbers. Salesforce includes this in their Unlimited tier at $350/user/month. Free in OutreachPro.

What the Dashboard Shows

  • Deal conversion funnel — count and value at each pipeline stage (Prospect → Won)
  • Best campaign — highest reply rate campaign and its performance
  • A/B test winners — all completed tests with the winning subject line
  • Overdue follow-ups — Memory CRM reminders that are past due
  • Activity breakdown — how many calls, emails, meetings logged across all leads
  • AI coaching insight — Claude's specific, actionable coaching recommendation based on actual pipeline data

Example Coaching Insight

"Your pipeline has 8 deals stuck in Contacted with no activity in 7+ days — prioritise follow-up calls this week. Your Tuesday email sends have a 38% open rate vs 21% on Monday, so reschedule next week's campaign launches to Tuesday mornings."

Using It as a Manager

  1. Open Analytics → Coaching Dashboard every Monday
  2. Check the funnel — where are deals stacking up? That stage needs attention.
  3. Review A/B test winners — update all active campaigns to use the winning subject
  4. Check overdue follow-ups — assign or action them before the day ends
  5. Read the AI insight — one specific thing to improve this week

API

GET /api/coaching/dashboard
→ { bestCampaign, completedAbTests[], funnel[], overdueFollowUps,
    activityBreakdown, coachingInsight }
