Email Template Library

Save, organise, and reuse your best-performing email templates. Generate AI variant versions with a different angle or tone. Track usage count per template. Built into OutreachPro — no separate tool needed.

Default Templates

  • Cold Outreach DE — German cold email with {{aiOpening}} token for personalisation
  • Follow-Up After No Reply — second touch for leads who didn't respond
  • Post-Demo Follow-Up — structured email after a demo with next-steps section

Creating a Template

  1. Go to Email Composer → Templates → New Template
  2. Write subject and body using {{tokens}} for personalisation
  3. Add tags for organisation (e.g. cold, follow-up, german, high-value)
  4. Save — template available across all campaigns and sequences

Available Tokens

{{name}}        — business name
{{city}}        — city
{{rating}}      — Google star rating
{{reviews}}     — review count
{{aiOpening}}   — Claude's personalised opening line
{{product}}     — your product name from workspace settings

AI Variant Generation

Click Generate Variant on any template. Specify a style: "shorter and more direct", "warmer tone", "focus on ROI", "more casual". Claude rewrites the template from a different angle while keeping all {{tokens}} intact. Use variants for A/B testing — run the original vs the variant to find the better performer.

API

GET    /api/templates            → list all templates
POST   /api/templates            → create {name, subject, body, tags[]}
POST   /api/templates/:id/variant → generate AI variant {style?}
DELETE /api/templates/:id        → delete
