Zoho CRM Integration

Zoho CRM is popular with SMEs and growing sales teams, especially in European and Indian markets. OutreachPro connects via OAuth2 to sync leads, create deals, and push bulk records efficiently — Zoho's API supports up to 100 records per request, making it the fastest CRM to bulk-sync to.

Getting Zoho OAuth Credentials

  1. Go to api-console.zoho.com (or .eu, .in depending on your region)
  2. Click Add Client → Self Client
  3. Copy your Client ID and Client Secret
  4. Click Generate Code → Scope: ZohoCRM.modules.ALL,ZohoCRM.settings.ALL
  5. Set time: 10 minutes → Generate
  6. Under Generate Access Token → paste the code → your account server region
  7. Copy the refresh_token from the response (save it — you can only see it once)
  8. In OutreachPro: Settings → CRM → Zoho → enter clientId, clientSecret, refreshToken, region → Test

Regions: com (US), eu (Europe), in (India), com.au (Australia), jp (Japan)

Bulk Sync — 100 Leads at Once

Zoho's API supports up to 100 records per request. OutreachPro automatically batches your leads into groups of 50 for reliable bulk sync. A list of 200 leads syncs in 4 API calls — much faster than HubSpot or Salesforce which process one record at a time.

Data Mapping

OutreachPro → Zoho CRM Lead
name        → Last_Name (+ First_Name split)
email       → Email
phone       → Phone
city        → City
website     → Website
category    → Description
source      → Lead_Source = "OutreachPro"
rating+reviews → Description field

Use Case: European SME Sales

Zoho CRM is particularly popular with German Mittelstand companies. If your prospects use Zoho, sync your OutreachPro leads directly into their Zoho instance when doing partnership or channel sales. Or use Zoho as your own CRM alongside OutreachPro — let OutreachPro handle prospecting and email, Zoho handle deal tracking and invoicing.

API

POST /api/crm/zoho/config    → {clientId, clientSecret, refreshToken, region}
POST /api/crm/zoho/test      → test connection
GET  /api/crm/zoho/leads     → list Zoho leads
POST /api/crm/zoho/leads     → create lead
POST /api/crm/zoho/deals     → create deal
POST /api/crm/zoho/sync      → bulk sync leads → Zoho (batches of 50)
