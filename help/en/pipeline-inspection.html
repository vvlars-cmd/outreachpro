Pipeline Inspection

A weekly snapshot of your pipeline showing exactly what changed: new deals entered, deals that moved forward, deals that stalled, and closed-won deals. Salesforce calls this "Pipeline Management" and includes it at their $175/user/month Enterprise tier. In OutreachPro it's a single API call.

What the Dashboard Shows

  • Active pipeline — total count and value of all open deals
  • Won this week — deals closed in the last 7 days + their value
  • New this week — deals that entered the pipeline in the last 7 days
  • Stalled — deals with no activity logged in 7+ days (danger zone)
  • By stage — breakdown of count and value at each pipeline stage

Stalled Deal Action Plan

Any deal with no activity in 7 days appears in the Stalled list. The Daily Task Queue also flags these with medium urgency. For each stalled deal:

  1. Check the Activity Timeline — what was the last touchpoint?
  2. Check Memory CRM — did the lead ask to be contacted later?
  3. Log a new activity: call, follow-up email, or LinkedIn message
  4. If no response within 7 more days: move to Lost and re-enrol in a win-back campaign

API

GET /api/pipeline/inspection
→ {
    summary: { activePipeline, totalPipelineValue, wonThisWeek, wonValue, newThisWeek, stalled },
    byStage: { Prospect: {count, value}, ... },
    stalledDeals: [{id, name, stage, value}],
    recentWins: [{id, name, value}]
  }
