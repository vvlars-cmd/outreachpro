Lead Segmentation

Automatically segment your lead database by category, city, rating tier, score tier, and status. Export any segment as CSV to run targeted campaigns, or use segment data to understand your pipeline composition.

Segment Types

  • By category — Fahrradhändler, E-Bike Händler, Fahrrad Werkstatt etc. — see which categories dominate your pipeline
  • By city — Köln, Düsseldorf, Bonn — which cities have the most leads? Which convert best?
  • By rating tier — Premium (4.5★+), Good (4.0–4.4★), Average (below 4★)
  • By score tier — Hot (70+), Warm (40–69), Cold (below 40)
  • By status — new, contacted, replied, converted

Using Segments for Targeted Campaigns

Example workflows:

  • Export all Premium-rated leads in Köln → personalised high-touch campaign
  • Export all Cold-scored leads with verified emails → different angle, lower effort
  • Export all "Fahrradhändler" category → category-specific subject line ("Ihr Fahrradgeschäft...")
  • Export all contacted-but-not-replied leads → stronger follow-up with video thumbnail

Exporting a Segment

  1. Go to Lead Finder → Segments
  2. Click on any segment to see the leads
  3. Click Export CSV — downloads with BOM for Excel compatibility
  4. Import the CSV into a targeted campaign

API

GET  /api/segments         → all segments auto-calculated
POST /api/segments/export  → {field, value, operator: 'eq'|'gte'|'lte'|'contains'} → CSV
