AI SDR — Autonomous Outreach Mode

The AI SDR is OutreachPro's most powerful feature. Claude acts as a virtual Sales Development Representative — analysing your leads, deciding who to target, writing personalised emails, and (in autopilot mode) sending them automatically.

Two Modes

  • Suggest — Claude proposes targets and email copy. You review and approve before anything sends. Recommended for new users.
  • Autopilot — Claude sends automatically up to your daily limit. No approval needed. Recommended once you've validated copy quality.

Setting Up AI SDR

  1. Go to Workspace → AI SDR
  2. Set daily limit (recommended: 10–20)
  3. Set target stages (e.g. leads with status "new")
  4. Set persona (e.g. "friendly professional", "direct and brief")
  5. Set language (de, en, fr, etc.)
  6. Click Save Config
  7. Click Run SDR

What Claude Does

For each eligible lead, Claude:

  1. Reads the lead's name, city, category, rating, review count, and website
  2. Writes a personalised subject line (not a template — unique to that lead)
  3. Writes a personalised opening line referencing their specific business details
  4. Provides reasoning for why this lead was chosen

Example Output

Lead: Fahrrad Müller, Köln, 4.8★, 342 reviews
Subject: "Ihre 342 Bewertungen sagen alles, Herr Müller"
Opening: "Mit 342 Google-Bewertungen und 4,8 Sternen haben Sie
offensichtlich eine loyale Kundschaft aufgebaut. Genau für
solche Betriebe haben wir den CycleWASH Pro entwickelt..."
Reasoning: "High review count + excellent rating = growing
business, strong social proof, likely investing in equipment"

API

GET  /api/ai-sdr/config  → current config
POST /api/ai-sdr/config  → update config
POST /api/ai-sdr/run     → run SDR session → {suggestions[]}
