Pipedrive Integration

Pipedrive is built for sales teams who manage deals in visual pipelines. OutreachPro connects to Pipedrive to sync your scraped leads as persons and organizations, push deals when prospects advance in your sequence, and log activities to keep your Pipedrive pipeline accurate.

Getting Your Pipedrive API Token

  1. Log in to Pipedrive → Click your avatar (top right) → Personal preferences
  2. Go to the API tab
  3. Copy your personal API token
  4. Your company domain is the subdomain from your Pipedrive URL: yourcompany.pipedrive.com → domain = yourcompany
  5. In OutreachPro: Settings → CRM → Pipedrive → paste token + domain → Test Connection

Data Mapping

OutreachPro Lead → Pipedrive
name             → Person name + Organization name
email            → Person email (label: work)
phone            → Person phone
city             → (in person notes)
website          → Organization website
status           → (deal stage when synced as deal)

How Sync Works

When you sync a lead to Pipedrive, OutreachPro creates two records:

  1. Organization — the business (e.g. "Fahrrad Müller Köln")
  2. Person — linked to the organization, with email and phone

You can then manually create a Deal linked to the person/org in Pipedrive, or use OutreachPro's deal creation API to auto-create it when a lead reaches "Proposal" stage.

Logging Activities from OutreachPro

When you send an email, OutreachPro can log a Pipedrive activity: type = "Email", subject = your email subject, due date = send date. This gives your Pipedrive timeline a complete record of all outreach. Enable in Settings → CRM → Pipedrive → Log emails as activities.

Use Case: Trade Show Follow-Up

After a trade show (e.g. Eurobike Köln), import the attendee list into OutreachPro → run a personalised follow-up sequence → sync all engaged leads (opened or replied) to Pipedrive → create deals for everyone who books a demo.

API

POST /api/crm/pipedrive/config         → {apiKey, domain}
POST /api/crm/pipedrive/test           → test connection
GET  /api/crm/pipedrive/persons        → list persons
POST /api/crm/pipedrive/persons        → create person
POST /api/crm/pipedrive/organizations  → create org
GET  /api/crm/pipedrive/deals          → list deals
POST /api/crm/pipedrive/deals          → create deal
POST /api/crm/pipedrive/activities     → log activity
POST /api/crm/pipedrive/sync           → sync leads → persons + orgs
