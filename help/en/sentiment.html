Email Sentiment Detection

When a lead replies, Claude reads their message and classifies it: Interested, Not Now, Objection, Meeting Request, or Unsubscribe. It extracts the specific objection type (budget, timing, competitor, no need), suggests the next action, and extracts a follow-up date if mentioned. The Unibox label updates automatically. Salesforce calls this "Einstein Email Insights" and includes it at $350/user/month. Free in OutreachPro.

Labels and What They Mean

  • Interested 🟢 — positive reply, wants to know more. Suggested action: book a demo immediately.
  • Meeting Request 🔥 — explicitly asking for a call or meeting. Urgency: hot. Send booking link immediately.
  • Not Now 🟡 — timing issue, not a no. Memory CRM creates a follow-up automatically.
  • Objection 🟠 — specific concern raised. Objection type extracted (budget, timing, competitor, no need).
  • Unsubscribe 🔴 — wants off the list. All sequences stop automatically via workflow.
  • Neutral ⚪ — informational reply, no clear buying signal.

Objection Handling Suggestions

  • Budget objection → send ROI calculation + financing options if available
  • Timing objection → Memory CRM follow-up for their stated date
  • Competitor objection → send comparison table + your differentiators
  • No need objection → re-qualify: ask about their specific pain point

Triggering Analysis

Analysis runs automatically when you click Analyse on any Unibox message, or call the API directly. The result updates the Unibox label, logs an activity to the lead's timeline, and triggers any matching workflows (e.g. "Unsubscribe → stop all sequences").

API

POST /api/inbox/analyse
  Body: { messageId, text, leadId }
  → { sentiment, intent, label, objection, urgency, followUpDate, suggestedAction }
