Workspace Settings

Configure your company details, product information, and event calendar. These settings power the AI personalisation — Claude uses your product name, pricing, and event dates when writing email opening lines.

Company Settings

  • Company name — appears in email signatures and AI-generated copy
  • Product name — the {{product}} token in email templates
  • Product description — used by Claude to write relevant opening lines
  • Website — linked in email signatures

Event Calendar

Add trade shows, demo days, and regional events. When an event is upcoming in the lead's city, OutreachPro automatically mentions it in the AI-generated opening: "We'll be in Köln for the Eurobike show next month — would love to show you the Pro Platinum in person."

White-Label Workspaces (Killer Feature #8)

Run OutreachPro as a white-label outreach agency. Each client gets their own isolated workspace with separate leads, campaigns, inbox, and CRM pipeline.

  1. Go to Workspace → New Workspace
  2. Enter client name, their sender email, and a brand colour
  3. OutreachPro creates isolated data files for this client
  4. Switch between workspaces using the top-bar selector

The master workspace view shows aggregate stats across all clients — total leads, active campaigns, pipeline value, and won revenue per workspace.

AI SDR Mode (Killer Feature #9)

Enable Autonomous SDR mode to let Claude decide who to target, when to reach out, and what to say — without any manual input.

  1. Go to Workspace → AI SDR
  2. Set mode: Suggest (Claude proposes, you approve) or Autopilot (sends automatically)
  3. Set daily limit (recommended: 10–20 for autopilot)
  4. Set target stages (e.g. "new" leads only)
  5. Click Run SDR — Claude analyses your leads and generates personalised subject + opening for each

In Suggest mode, you get a review screen before anything sends. In Autopilot, Claude sends directly up to the daily limit.
