CPQ — Product Catalogue & Pricing Calculator

Configure your products, prices, and discount rules once. When generating a proposal or quote, OutreachPro calculates the total automatically — applying volume discounts, deal-specific discounts, and a global discount if needed. Salesforce CPQ is a $75/user/month add-on. In OutreachPro it's built in.

Default Product Catalogue (CycleWASH)

  • CycleWASH Pro Platinum — €38,000 base price. 5% volume discount from qty 2, 10% from qty 5.
  • CycleWASH Mini Platinum — €27,500 base price. 5% volume discount from qty 2.
  • Extended Warranty (5yr) — €3,500. No volume discount.
  • Installation & Training — €1,200. No volume discount.

Configuring Products

  1. Go to Settings → Product Catalogue
  2. Click Add Product
  3. Set name, base price, currency, category, and discount rules
  4. Discount rules: [{ minQty: 2, discountPct: 5 }, { minQty: 5, discountPct: 10 }]

Building a Quote

  1. Go to CRM → Deal → Build Quote or Proposal Generator
  2. Select products and quantities
  3. Add a deal-specific manual discount if applicable
  4. OutreachPro calculates: line totals, applied discounts, subtotal, and final total
  5. Total feeds into the Proposal Generator automatically

Discount Priority

Volume discount (from rules) + Manual discount (per line) + Global discount
= Total discount per line item (capped at 40%)

API

GET    /api/products              → list product catalogue
POST   /api/products              → add product
POST   /api/cpq/calculate
  Body: { items: [{productId, qty, manualDiscountPct}], globalDiscountPct }
  → { lineItems[], subtotal, total, currency, validDays }
DELETE /api/products/:id          → remove product
