A/B Subject Line Testing

Test two subject line variants against each other in any campaign. OutreachPro tracks opens per variant and automatically promotes the winner — no manual analysis needed.

Creating an A/B Test

  1. Open any Campaign and click A/B Test
  2. Enter Subject A and Subject B
  3. Set the split (default: 50% each)
  4. Set auto-promote threshold (default: 50 sends — after 50 total emails sent, the winner is auto-promoted)
  5. Click Start Test

How Auto-Promotion Works

As emails send, OutreachPro records opens per variant. When the total send count reaches the threshold (e.g. 50), it calculates the open rate for each:

Subject A: 22 sends, 8 opens → 36.4% open rate
Subject B: 28 sends, 7 opens → 25.0% open rate
→ Winner: Subject A (promoted automatically for remaining sends)

Subject Line Best Practices

  • Test one variable at a time — different length, personalisation, or angle
  • Subject A: question format — "Quick question about {{name}}'s workshop?"
  • Subject B: observation format — "Noticed {{name}}'s {{rating}}★ rating on Google"
  • Avoid ALL CAPS, excessive punctuation, and spam trigger words
  • Keep under 50 characters for full mobile display

API

GET    /api/ab-tests                    → list all tests
POST   /api/ab-tests                    → create test
POST   /api/ab-tests/:id/record         → record event {variant, event}
DELETE /api/ab-tests/:id                → delete test
