Salesforce Integration

Connect OutreachPro to Salesforce to push scraped leads as Salesforce Leads, convert hot prospects into Opportunities, and run SOQL queries directly from OutreachPro. Supports both Username-Password OAuth2 (simplest for self-hosted) and direct access token.

Getting Salesforce Credentials

Use the Username-Password OAuth2 flow for self-hosted OutreachPro:

  1. In Salesforce: Setup → App Manager → New Connected App
  2. Check Enable OAuth Settings
  3. Callback URL: http://localhost:3000/api/crm/salesforce/callback
  4. Selected OAuth Scopes: api, refresh_token
  5. Save → wait 10 minutes for propagation
  6. Copy Consumer Key (= Client ID) and Consumer Secret
  7. Your password for the API: your_sf_password + your_security_token (concatenated, no space)
  8. In OutreachPro: Settings → CRM → Salesforce → fill in all fields → Test

Alternatively, if you already have an access token from Salesforce Workbench or Postman, you can enter it directly in the Access Token field along with your Instance URL.

Lead Data Mapping

OutreachPro → Salesforce Lead
name        → LastName + FirstName (split by space)
email       → Email
phone       → Phone
city        → City
website     → Website
category    → Description (with rating + review count)
source      → LeadSource = "OutreachPro"
status      → Status = "Open - Not Contacted"

SOQL Queries from OutreachPro

Run any SOQL query against your Salesforce org directly from OutreachPro's interface. Useful for pulling back converted leads, checking deal progress, or building reports. Go to CRM → Salesforce → Query.

-- Example queries:
SELECT Id, Name, Email, Status FROM Lead WHERE Status = 'Working - Contacted'
SELECT Id, Name, Amount, StageName FROM Opportunity WHERE StageName != 'Closed Lost'
SELECT Id, Name, Account.Name FROM Contact WHERE CreatedDate = LAST_N_DAYS:30

Use Case: Enterprise Sales Team

Your Salesforce is the system of record. Use OutreachPro for prospecting and outreach — it's faster, cheaper, and more personalised than Salesforce's native outreach tools. When a lead is qualified, sync them to Salesforce as a Lead. When they convert, create an Opportunity from OutreachPro's CRM Proposal generator and push it straight to Salesforce.

API

POST /api/crm/salesforce/config       → {clientId, clientSecret, username, password, instanceUrl}
POST /api/crm/salesforce/test         → test connection
POST /api/crm/salesforce/leads        → create Salesforce Lead
POST /api/crm/salesforce/opportunities → create Opportunity
POST /api/crm/salesforce/query        → {query: "SELECT ..."} → SOQL results
POST /api/crm/salesforce/sync         → sync leads → SF Leads
