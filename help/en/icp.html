ICP Builder — Ideal Customer Profile

Define your ideal customer profile once and OutreachPro automatically finds and imports matching leads on a weekly schedule. No more manual scraping.

Setting Up Your ICP

  1. Go to Settings → ICP Builder
  2. Add your target cities (comma-separated)
  3. Add search terms (what you'd type into Google Maps)
  4. Set minimum rating (default: 4.0)
  5. Set minimum reviews (default: 10)
  6. Set max leads per run (default: 50)
  7. Click Save ICP
  8. Click Run Now to test it immediately, or enable Weekly Auto-Run

Example ICP — Bike Dealer

Cities: Köln, Düsseldorf, Bonn, Aachen, Leverkusen, Wuppertal
Terms: Fahrradhändler, E-Bike Händler, Fahrrad Service
Min Rating: 4.0
Min Reviews: 15
Max per run: 50

This runs 6 cities × 3 terms = 18 scrape jobs, importing up to 50 top-scored leads per week. Over 4 weeks that's 200 pre-qualified leads automatically in your pipeline.

Auto-Import Logic

Only leads scoring above your threshold (default: 60/100) are imported. Leads already in your database (matched by name + city) are not duplicated. New leads land in the Lead Finder with status "new" and source "ICP auto-import".

API

GET  /api/icp         → current ICP config
POST /api/icp         → update ICP config
POST /api/icp/run     → trigger immediate run
