CRM Integrations — Overview

OutreachPro connects directly to the five most popular CRM tools used by marketers and sales teams. Once configured, you can push leads, sync deals, create contacts, log notes, and keep everything in sync — without ever leaving OutreachPro.

Supported CRMs

  • 🟠 HubSpot — contacts, companies, deals, notes (API v3)
  • 🔵 Pipedrive — persons, organizations, deals, activities (API v2)
  • ☁️ Salesforce — leads, opportunities, contacts, SOQL queries (REST API v59)
  • 🟢 Zoho CRM — leads, deals, contacts, bulk sync (API v6)
  • ⚪ Notion — CRM-as-database, pages, properties (REST API)

Quick Setup Flow

  1. Go to Settings → CRM Integrations
  2. Click the CRM you want to connect
  3. Paste your API key / credentials
  4. Click Test Connection — green = connected
  5. Click Sync Leads to push your OutreachPro leads to that CRM

One-Button Sync All

Go to Settings → CRM Integrations → Sync All CRMs. OutreachPro pushes your leads to every connected CRM simultaneously. Useful after a big Apify scrape — import the leads, score them, then sync the top-scoring ones to all your CRMs in one click.

Sync Log

Every sync is logged with: CRM name, number synced, number failed, and timestamp. Go to Settings → CRM Integrations → Sync History to see the full log. Useful for auditing what's been pushed where.

API — Status Check

GET /api/crm/status
→ {
    hubspot:    { configured: true },
    pipedrive:  { configured: false },
    salesforce: { configured: true },
    zoho:       { configured: false },
    notion:     { configured: true }
  }

POST /api/crm/sync-all
  Body: { limit: 25, leadIds?: [] }
  → { results: { hubspot: {...}, pipedrive: {...}, ... } }

GET /api/crm/sync-log
  → [{crm, synced, failed, ts}]
