LinkedIn Outreach

Add LinkedIn connection requests and messages as steps in your outreach sequences. Claude generates personalised connection notes for each lead — staying within LinkedIn's 300-character limit for connection requests.

Adding LinkedIn Steps

  1. Go to Sequences and open any sequence
  2. Click Add Step → LinkedIn
  3. Choose type: Connection Request or Message
  4. Click AI Generate — Claude writes a personalised note
  5. Review and edit if needed
  6. Set delay (e.g. Day 2 — send connection request 2 days after first email)

Recommended Multichannel Sequence

Day 0  — Email: personalised pitch
Day 2  — LinkedIn: connection request (personalised note)
Day 5  — Email: follow-up with case study
Day 8  — LinkedIn: message to connected lead
Day 14 — Email: final follow-up

This multichannel approach generates 3–4× more replies than email-only sequences.

Connection Request Templates

OutreachPro ships with 3 default LinkedIn templates. Add your own at Settings → LinkedIn Templates. Claude uses the template as a starting point and personalises it for each lead.

Important Notes

  • LinkedIn connection requests must be sent manually — LinkedIn's API prohibits automated connection requests
  • OutreachPro generates the text and notifies you when it's time to send
  • Connection requests: max 300 characters
  • Messages to connections: max 500 characters
  • LinkedIn limits ~100 connection requests per week — pace yourself

API

GET  /api/linkedin/templates         → list templates
POST /api/linkedin/templates         → create template
POST /api/linkedin/generate          → generate message for lead
  Body: { lead, templateId, type: 'connection'|'message' }
  → { message, charCount }
