Sales Goals & Progress Tracker

Set monthly targets for revenue, leads, demos, and emails sent. OutreachPro tracks real-time progress against each goal and flags whether you're on pace. Goals feed into the Coaching Dashboard and Daily Task Queue.

Default Goals

  • Monthly Revenue — sum of deals marked Won this month vs target
  • New Leads per Month — leads added to the database this month
  • Demos Booked — meeting_booked activity events this month
  • Emails Sent — email_sent activity events this month

On-Track Logic

OutreachPro compares your current % of goal to how far through the month you are. If you're on Day 15 (50% through the month) and you've hit 56% of your revenue goal — you're on track. If you've only hit 30% — you need to accelerate.

Projected End of Month

OutreachPro extrapolates your current pace to project your end-of-month result: "At current pace, you'll close €56,000 by month end (target: €50,000)". This lets you decide whether to push harder or adjust targets early.

Adding Custom Goals

  1. Go to Analytics → Goals
  2. Click New Goal
  3. Set name, type (revenue/leads/meetings/emails_sent), period, and target
  4. Save — progress calculates automatically

API

GET    /api/goals              → all goals with current progress + projections
POST   /api/goals              → create {name, type, period, target, currency?}
DELETE /api/goals/:id          → delete goal
