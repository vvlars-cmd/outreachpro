Sales Playbooks

Battle cards for every common sales scenario — structured step-by-step guides for handling objections, opening cold calls, running demos, and closing deals. Salesforce Sales Enablement charges $150+/month for this. Built into OutreachPro.

Default Playbooks

  • Budget Objection — acknowledge, reframe to ROI, offer financing options
  • Timing Objection — qualify, plant a seed, lock in a future date
  • Competitor Objection — never badmouth, understand what they value, differentiate
  • Cold Call Opener — permission-based open, hook with their reviews, bridge to problem
  • Demo Closing — recap highlights, address hesitation, lock in next step

Using a Playbook

Each playbook has:

  • Steps — the exact sequence of what to say, in order
  • Talking points — supporting facts and proof points to have ready
  • Key question — the one question that moves the conversation forward

Access playbooks from the Call Script screen — when a lead's sentiment analysis detects an objection type, the matching playbook appears automatically.

Creating a Custom Playbook

  1. Go to Settings → Playbooks → New Playbook
  2. Set name, trigger keyword (maps to objection types), and category
  3. Add steps, talking points, and the key question
  4. Save — playbook appears automatically when the trigger is detected

API

GET  /api/playbooks              → list all playbooks
POST /api/playbooks              → create playbook
GET  /api/playbooks/:trigger     → get playbook by trigger (budget|timing|competitor|cold_call|post_demo)
