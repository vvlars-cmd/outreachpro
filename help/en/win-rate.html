Win Rate Analytics

Track the percentage of closed deals that were won vs lost — broken down by time period, campaign source, and monthly trend. Identify which campaigns generate the most closeable leads, and where deals are dying.

Key Metrics

  • Win rate — won ÷ (won + lost) × 100. Industry benchmark: 20–30% for cold outreach.
  • Average deal size — mean value of won deals. Track this over time — rising = upselling working.
  • Win rate by campaign — which outreach source closes best. Allocate more time there.
  • Monthly trend — are you winning more or fewer deals each month?

Time Periods

Use the period filter: 30d, 90d (default), 180d, 365d. Shorter periods show recent performance; longer periods show seasonal patterns and trends.

Using Win Rate to Improve

  • Win rate below 15%? Your leads aren't qualified enough — tighten ICP criteria or score higher
  • Win rate above 40%? You're under-prospecting — add more volume
  • One campaign source has 2× win rate of others? Move budget there
  • Win rate dropping month-over-month? Review the Coaching Dashboard — something changed

API

GET /api/analytics/win-rate?period=90d
→ { winRate, won, lost, total, avgDealSize, totalWonValue, byCampaign, monthlyTrend }
