White-Label Client Workspaces

Run OutreachPro as an outreach agency. Each client gets a completely isolated workspace — their own leads, campaigns, inbox, CRM, and analytics — all from one OutreachPro instance.

Creating a Workspace

  1. Go to Workspace → Manage Workspaces
  2. Click New Workspace
  3. Enter the client name and their sender email
  4. Pick a brand colour (shows in the workspace switcher)
  5. Click Create

OutreachPro creates a dedicated data directory for this workspace. All leads, campaigns, and deals created while in this workspace are stored separately — completely isolated from other clients.

Switching Workspaces

Use the workspace selector in the top bar. The current workspace name and colour are shown. Switching instantly loads that workspace's data — leads, campaigns, CRM, and analytics.

Master Dashboard

The default workspace shows aggregate stats across all workspaces: total leads, active campaigns, pipeline value, and won revenue per client. Use this to monitor all clients at a glance.

Workspace Stats API

GET /api/workspaces              → list all workspaces
POST /api/workspaces             → create workspace
GET /api/workspaces/:id/stats   → leads, campaigns, deals, inbox count
DELETE /api/workspaces/:id      → delete workspace

Agency Pricing Model

With white-label workspaces, you can charge clients €500–2000/month for managed outreach — while your actual cost is OutreachPro's API bills (~€50–100/month). The workspace isolation means each client sees only their data, making the product fully white-label ready.
