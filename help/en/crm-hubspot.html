HubSpot Integration

Connect OutreachPro to HubSpot to sync leads as contacts, create deals automatically when meetings are booked, log outreach activity as notes, and keep your HubSpot pipeline up to date without manual data entry.

Getting Your HubSpot API Key

  1. Log in to HubSpot → Click your account name (top right) → Profile & Preferences
  2. Go to Integrations → Private Apps
  3. Click Create a private app
  4. Give it a name (e.g. "OutreachPro Sync")
  5. Under Scopes, enable: crm.objects.contacts.write, crm.objects.deals.write, crm.objects.notes.write, crm.objects.contacts.read, crm.objects.deals.read
  6. Click Create app → copy the access token
  7. In OutreachPro: Settings → CRM → HubSpot → paste token → Test Connection

What Gets Synced

  • Lead name → HubSpot Contact (firstname + lastname split automatically)
  • Lead email, phone, website → Contact properties
  • Lead city → Contact city property
  • Lead status → hs_lead_status = "NEW"
  • Company name → HubSpot Company (associated to contact)

Automatically Create Deals When Meetings Are Booked

When a lead books a meeting via the Meeting Scheduler, OutreachPro can automatically create a HubSpot deal in "Appointment Scheduled" stage. Configure this in Settings → Scheduler → On booking: Create HubSpot deal.

Log Outreach as HubSpot Notes

After sending a campaign email, OutreachPro can log it as a HubSpot note on the contact record — including subject, send time, and campaign name. Enable this in Settings → CRM → HubSpot → Log sent emails as notes.

HubSpot Deal Stages

OutreachPro maps its CRM stages to HubSpot deal stages automatically:

OutreachPro Stage → HubSpot Stage
Contacted         → appointmentscheduled
Replied           → qualifiedtobuy
Meeting           → presentationscheduled
Proposal          → decisionmakerboughtin
Won               → closedwon
Lost              → closedlost

Use Case: CycleWASH Pipeline

Scrape 50 bike dealers with Apify → Score leads → Sync top 20 to HubSpot → Run OutreachPro campaign → When a dealer replies, automatically move their HubSpot contact to "Replied" stage → When they book a demo, auto-create a deal in "Appointment Scheduled" with €38,000 value.

API

POST /api/crm/hubspot/config    → {apiKey, portalId}
POST /api/crm/hubspot/test      → test connection
GET  /api/crm/hubspot/contacts  → list contacts
POST /api/crm/hubspot/contacts  → create contact
GET  /api/crm/hubspot/deals     → list deals
POST /api/crm/hubspot/deals     → create deal
POST /api/crm/hubspot/notes     → add note to contact/deal
POST /api/crm/hubspot/sync      → sync OutreachPro leads → HubSpot
