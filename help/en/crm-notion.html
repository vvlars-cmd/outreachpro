Notion CRM Integration

Many teams use Notion as their CRM — a database where each row is a lead or deal, with custom properties for status, rating, city, website, and notes. OutreachPro connects directly to your Notion CRM database to sync leads, update statuses, and keep everything in one place.

Setting Up Your Notion CRM Database

First, create a Notion database with these properties (OutreachPro will map to them automatically by name):

Name     — Title (required)
Email    — Email
Phone    — Phone number
Company  — Text
City     — Text
Status   — Select (options: New, Contacted, Replied, Meeting, Won, Lost)
Rating   — Number
Website  — URL
Category — Select
Notes    — Text

You can add extra properties — OutreachPro will only write to the ones it knows about.

Creating a Notion Integration Token

  1. Go to notion.so/my-integrations → New integration
  2. Name it "OutreachPro", select your workspace
  3. Capabilities: Read content, Update content, Insert content
  4. Copy the Internal Integration Token (starts with secret_)
  5. In your Notion database: click ··· → Add connections → select your OutreachPro integration
  6. Copy the Database ID from the database URL: notion.so/workspace/DATABASE_ID?v=...
  7. In OutreachPro: Settings → CRM → Notion → paste token + database ID → Test

How It Works

Each OutreachPro lead becomes a page (row) in your Notion database. The lead's name, email, phone, city, rating, website, and category are mapped to Notion properties. When a lead's status changes in OutreachPro, you can sync the update back to Notion to keep both in sync.

Use Case: Solo Founder / Small Team

If you already live in Notion for notes, tasks, and planning — use it as your CRM too. OutreachPro handles all the outreach automation; Notion is your relationship memory. After every call, add notes directly in Notion. Use Notion's filter views to see all leads in "Meeting" stage, all leads from Köln, or all leads with rating above 4.5.

Use Case: Content + Outbound Team

Marketing team uses Notion for content planning. Sales team uses OutreachPro for outreach. Sync OutreachPro leads to a shared Notion database — marketing sees who's being targeted, adds context (blog posts the lead might like, relevant case studies), and both teams stay aligned without Slack threads.

API

POST  /api/crm/notion/config        → {apiKey, databaseId}
POST  /api/crm/notion/test          → test connection
GET   /api/crm/notion/pages         → list database entries
POST  /api/crm/notion/pages         → create page (lead entry)
PATCH /api/crm/notion/pages/:pageId → update status/notes
POST  /api/crm/notion/sync          → sync leads → Notion database
