Compositor de correos HTML

El compositor le permite crear correos electrónicos HTML ricos con personalización por IA.

Tokens de variables

Use {{name}}, {{company}}, {{city}}, {{product}} para personalización automática.

Asistente IA

Seleccione un cliente potencial, haga clic en "Generar con IA" y Claude creará una línea de apertura personalizada basada en los datos de Apify.

Personalización masiva

Haga clic en "Personalizar todos" para generar aperturas únicas para cada cliente potencial con email.
