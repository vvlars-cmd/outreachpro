Getting Started

5-minute quickstart

From zero to your first campaign in 5 minutes.

1
Start the app
Make sure Docker Desktop is running, then:
cd ~/outreachpro-app
docker compose up --build -d
open http://localhost:3000
✓ Health check: localhost:3000/health
2
Add your API keys
Go to Settings in the left sidebar. Add your keys:
• Anthropic API key — for AI personalisation (console.anthropic.com)
• Apify token — for lead scraping (console.apify.com)
• Gmail credentials — for sending (optional, Google Cloud Console)
⚙ Settings → Integrations
3
Scrape your first leads
Go to Apify Scraper. Enter a city (e.g. "Köln") and search term (e.g. "Fahrradhändler"). Click Scrape Leads. Watch leads appear in real-time.
🗺 Apify Scraper → Scrape Leads
4
Personalise with AI
Go to Email Composer. Select a lead from the dropdown. Click Generate with AI. Claude writes a personalised opening line using the lead's business data (category, rating, reviews, website).
✦ Email Composer → Generate with AI
5
Create and launch a campaign
Click + Campaign in the top bar. Set your daily send limit to 20–40. Enable Stop on Reply. Go to Sequences and add 2–3 email steps with 3-day delays. Launch.
📧 Daily limit 20–40 · Stop on Reply ON
Concepts

How OutreachPro works

The full outreach flow from lead discovery to closed deal:

Apify scrape → leads.json → AI personalise → campaign → sequence ↓ ↓ Gmail draft ←── Unibox ←── replies ←── emails sent ←── warmup

Data flow

All data is stored as JSON files on your Docker volume (/app/data/). Nothing leaves your server unless you explicitly use the Gmail or Anthropic APIs. The .gitignore excludes all data files from version control.

🐳
OutreachPro runs in a single Docker container (Node 20 Alpine). Your data persists in a Docker volume. To back up: docker cp outreachpro:/app/data ./backup

Feature

Apify Google Maps Scraper

Finds local businesses on Google Maps and extracts contact details. Requires an Apify token (free tier available at console.apify.com).

How to scrape

  1. Go to Apify Scraper in the nav
  2. Enter target city (e.g. Köln)
  3. Enter search term (e.g. Fahrradhändler)
  4. Set minimum rating (3.5★ recommended)
  5. Click Scrape Leads
💡
Best search terms for bike dealers in Germany: Fahrradhändler, E-Bike Händler, Fahrradgeschäft, Zweiradhandel

What gets extracted

Name · Email · Phone · Website · Category · Rating · Review count · Full address · All imported and deduplicated automatically.

Feature

Campaigns

A campaign sends a sequence of emails to your leads, stopping automatically when they reply.

Campaign settings

SettingWhat it doesRecommended
nameYour campaign nameInclude date + target city

Performance targets

  • Open rate > 40% — good subject line
  • Reply rate > 10% — good email body + AI personalisation
  • Bounce rate < 3% — clean email list
Feature

Sequences

Multi-step email sequences automatically send follow-ups after a set number of days if the lead hasn't replied.

Example 3-step sequence
Step 1 (day 0): Intro — product overview + {{aiOpening}} Step 2 (day 3): Follow-up — "did you get a chance to read my email?" Step 3 (day 7): Final — case study or social proof + soft close
💡
3 steps is the sweet spot. Studies show reply rate peaks at step 2 for cold email. Going beyond 4 steps hurts sender reputation.

Email Warmup

Builds your sender reputation before launching campaigns. Slow-ramps from 5 to 50 emails/day over 14 days.

Reputation score guide

ScoreStatusAction

⚠

AI Features

AI Personalisation

Uses Claude Sonnet to write a unique opening sentence per lead, based on real business data.

POST /api/ai/personalise { "lead": { "name": "Rad & Tat Köln", "city": "Köln", "category": "Fahrradhändler", "rating": 4.8, "reviewsCount": 320, "website": "https://rad-und-tat.de" }, "language": "de", "productContext": "bicycle washing machine" }

Returns: { "opening": "Mit 320 Bewertungen und 4,8 Sternen zeigt Rad & Tat…" }

✦
The AI reads the lead's review count, rating, category, and website to craft something genuinely specific. Personalised emails get 3× more replies than generic templates.
AI Features

AI Copilot

Ask Claude anything about your outreach strategy. Provides advice based on your current campaign data.

POST /api/ai/copilot { "message": "How can I improve my open rate?", "context": { "openRate": 22, "campaigns": 3 } }

Use it to: improve subject lines, fix low open rates, write better follow-ups, analyse why a campaign underperformed.

AI Features

AI Reply Labels

Automatically classifies inbox replies into: interested, meeting, not_now, other.

POST /api/inbox/label { "id": "msg123", "label": "interested" }

The AI reads the full reply body and assigns the most appropriate label. You can override any label manually.

Troubleshooting

Common issues

App won't start

cd ~/outreachpro-app docker compose down docker compose up --build -d docker compose logs -f # check for errors

Buttons not working

Press Cmd+Shift+R (hard refresh). If still broken, click the Hard Reset button at the bottom of the sidebar.

Language not switching

The i18n fix requires the latest server.js. Check version badge shows v2.5 or later. If not, rebuild Docker with the latest server.js.

AI not personalising

Check ANTHROPIC_API_KEY in Settings → Integrations. Verify the key starts with sk-ant-api03-.

Scraper returning no results

Check APIFY_TOKEN in Settings. Make sure the token is active at console.apify.com. Free tier allows ~200 results/month.

Emails going to spam

1. Run email warmup for 14+ days before sending campaigns. 2. Check domain SPF, DKIM, DMARC records. 3. Use the Inbox Tester to check your spam score. 4. Start with a daily limit of 20 emails max.

FAQ

Frequently asked questions

Is my data safe?

Yes. OutreachPro is fully self-hosted. Your leads, campaigns, and emails never leave your server. Only the API calls to Anthropic and Apify leave your machine — these send only the specific data needed for that request (a lead's business info for personalisation, or a search query for scraping).

How much does it cost to run?

Typical monthly API costs: Anthropic ~€10–30 (personalising 1,000–3,000 leads), Apify ~€5–15 (scraping 500–2,000 leads). Total: €15–50/month vs €99+/seat/month for Apollo or Instantly.

Is cold email legal in Germany/EU?

B2B cold email is permitted under GDPR legitimate interest, as long as you email business addresses found via public sources (like Google Maps) and include an unsubscribe option. Always consult your legal counsel for your specific situation.

Can I use it for multiple companies?

Yes — the Workspace module lets you configure different company profiles. The Agency pricing plan ($349/mo) is designed for managing multiple client workspaces.

How do I update to a new version?

cd ~/outreachpro-app git pull origin main docker compose up --build -d
